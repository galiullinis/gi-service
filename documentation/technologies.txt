Для разработки веб-сервиса gi-service, выбраны следующие технологии:
1. Frontend часть будет реализована с помощью языков CSS и HTML. Она будет отправлять и принимать HTTP запросы.
2. Взаимодействие с backend частью, будет осуществляться с помощью шаблонизатора "mustache".
3. Backend часть будет реализована на языке Java с помощью Spring Framework.
    Преимущества Spring:
    1. Минимизирует время разработки и развертки веб-приложения и поднимает производительность
    2. Избегает  написание многих кодов прототипа благодаря Annotations и конфигурации XML
    3. Множество готовых фреймворков для облегчения работы программиста
    4. Легко позволяет взаимодействовать с приложениями Spring Boot c системами Spring таких как Spring JDBC, Spring ORM, Spring Data, Spring Security и т.д
    5. Обеспечивает встроенный Web-server как Tomcat, Jetty .... чтобы быстро и легко разворачивать и тестировать веб-приложения
4. Веб-сервером, обрабатывающим HTTP запросы, будет Tomcat(стандартный веб-сервер spring framework)
5. Для сборки проекта, взят фреймворк Maven
